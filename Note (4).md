## 程序的组织结构
计算机的流程控制：顺序结构、选择结构（if）、循环结构（while、for-in）
## 对象的布尔值
获取对象的布尔值，使用内置函数bool()
以下对象的布尔值均为False
- False
- 数值0
- None
- 空字符串
- 空列表 \[]或list() 
- 空元组 \()或tuple()
- 空字典 \{}或dict()
- 空集合 \set()
## 分支结构
### 单分支结构
**==if 条件表达式:
	条件执行体==**
### 双分支结构
**==if 条件表达式:
	条件执行体1
else:
	条件执行体2==**
### 多分支结构
**==if 条件表达式1:
	条件执行体1
elif 条件表达式2:
	条件执行体2
elif 条件表达式N:
	条件执行体N
else:
	条件执行体N+1==**
### 条件表达式（简写）
==**x if 条件判断 else y==**
条件判断为True，执行x；反之，执行y
```python
print(str(num1)+'大于等于'+str(num2) if num1 >= num2 else str(num1)+'小于'+str(num2))
```
